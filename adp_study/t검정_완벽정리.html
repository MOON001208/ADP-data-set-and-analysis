T-검정(t-test) 완벽 정리 - 개념, R/Python 코드 및 해석

지문만 보고 1초 만에 파악하는 T-검정(t-test) 완벽 가이드

T-검정(t-test)은 모집단의 분산을 알지 못할 때, 표본 평균을 통해 집단의 평균을 비교하고 가설을 검정하는 방법입니다. ADP 실기 시험에서는 지문에 등장하는 ‘비교 대상의 수’와 ‘독립성 여부’에 따라 3가지 검정 중 하나를 정확히 선택해야 합니다.

🎯 [핵심] ADP 지문 키워드로 보는 t-검정 선택 가이드

💡 방향성(대립가설) 설정 팁: * “차이가 있는가?”, “같은가/다른가?” -> 양측 검정(Two-sided) (기본값) * “~가 더 큰가(높은가)?”, “~가 더 작은가(낮은가)?” -> 단측 검정(Greater / Less) 설정 필수!

모든 t-검정은 ‘정규성 가정’을 전제로 합니다. 정규성을 만족하지 못하면 비모수 검정을 사용해야 하므로, 답안 작성 시 반드시 모수/비모수 선택의 근거를 코드로 보여주어야 합니다.

1. 단일 표본 t-검정 (One-sample t-test)

특정 집단의 표본 평균이, 사전에 알려진 통상적인 평균값(기준값)과 일치하는지 확인합니다. 독립변수 0개, 종속변수 1개일 때 사용합니다.

  • 상황: A 커피숍에서 판매하는 아메리카노의 평균 용량이 300ml라고 홍보하고 있다. 실제로 30잔을 무작위로 추출하여 측정한 결과, 이 집단의 평균 용량이 300ml와 차이가 있는지 검정하시오.
  • 가설:
    • \(H_0\) (귀무가설): 아메리카노의 평균 용량은 300ml 이다.
    • \(H_1\) (대립가설): 아메리카노의 평균 용량은 300ml가 아니다.

🐍 Python 코드

import numpy as np
import scipy.stats as stats

# 1. 데이터 세팅 (평균 295, 샘플 30개 추출)
np.random.seed(42)
coffee_vol = np.random.normal(loc=295, scale=10, size=30) 

# 2. 정규성 검정 (Shapiro-Wilk)
stat, p_val = stats.shapiro(coffee_vol)
print(f"[정규성 검정] p-value: {p_val:.4f}") 
[정규성 검정] p-value: 0.6868
# 결과: p > 0.05 이면 정규분포를 따름 -> t-test 진행
# (불만족 시 비모수 검정: stats.wilcoxon(coffee_vol - 300) 수행)

# 3. 단일 표본 t-검정
t_stat, t_pval = stats.ttest_1samp(coffee_vol, popmean=300)
print(f"[단일표본 t-검정] p-value: {t_pval:.4f}")
[단일표본 t-검정] p-value: 0.0002

📝 [결론 해석 모범 답안 작성법]

Shapiro-Wilk 검정 결과 p-value가 유의수준 0.05보다 커 정규성 가정을 만족한다. 이에 단일 표본 t-검정을 수행한 결과, p-value가 0.003으로 도출되었다. 유의수준 0.05 하에서 귀무가설을 기각하므로, A 커피숍의 아메리카노 평균 용량은 300ml라고 볼 수 없으며 홍보 내용과 유의미한 차이가 존재한다.

2. 독립 표본 t-검정 (Independent Two-sample t-test)

서로 다른 독립적인 2개 집단의 평균이 같은지를 검정합니다. 이 검정은 유일하게 정규성 검정 후 ‘등분산성 검정’을 추가로 거쳐야 합니다.

  • 상황: 남학생 30명과 여학생 30명을 대상으로 체육 실기 시험을 보았다. 성별에 따른 실기 점수 평균에 차이가 있는지 검정하시오.
  • 가설:
    • \(H_0\) (귀무가설): 남학생과 여학생의 평균 점수에는 차이가 없다.
    • \(H_1\) (대립가설): 남학생과 여학생의 평균 점수에는 차이가 있다.

🐍 Python 코드

import numpy as np
import scipy.stats as stats

np.random.seed(123)
score_m = np.random.normal(loc=85, scale=5, size=30)
score_f = np.random.normal(loc=78, scale=5, size=30)

# 1. 정규성 검정
print("남학생 정규성 p: ", stats.shapiro(score_m).pvalue)
남학생 정규성 p:  0.3508744273313644
print("여학생 정규성 p: ", stats.shapiro(score_f).pvalue)
여학생 정규성 p:  0.971662476742308
# 둘 중 하나라도 불만족하면 비모수(stats.mannwhitneyu(score_m, score_f))로 이동!

# 2. 등분산성 검정 (Levene/Bartlett) - H0: 두 집단의 분산이 같다.
stat, levene_p = stats.levene(score_m, score_f)
print(f"[등분산성 검정] p-value: {levene_p:.4f}")
[등분산성 검정] p-value: 0.9592
# 3. 독립 표본 t-검정 
# p>0.05(등분산 만족) -> equal_var=True (Student's t-test)
# p<0.05(등분산 불만족) -> equal_var=False (Welch's t-test)
t_stat, t_pval = stats.ttest_ind(score_m, score_f, equal_var=True)
print(f"[독립표본 t-검정] p-value: {t_pval:.4f}")
[독립표본 t-검정] p-value: 0.0001

3. 대응 표본 t-검정 (Paired t-test)

동일한 객체(사람, 기계 등)에 대해 두 가지 처리를 한 후, ‘처치 전’과 ’처치 후’ 측정값 간에 차이가 생겼는지 검정합니다. 독립 표본이 아니라는 점이 핵심입니다.

  • 상황: 고혈압 환자 20명에게 새로운 혈압 강하제를 투여하기 전과 투여한 후의 혈압을 각각 측정하였다. 이 약이 혈압 하강에 유의미한 효과가 있는지 검정하시오.
  • 가설 (단측 검정):
    • \(H_0\) (귀무가설): 투여 후 혈압 - 투여 전 혈압 \(\ge 0\) (효과가 없다)
    • \(H_1\) (대립가설): 투여 후 혈압 - 투여 전 혈압 \(< 0\) (투여 후 혈압이 더 낮아졌다 / 효과가 있다)

🐍 Python 코드

import numpy as np
import scipy.stats as stats

np.random.seed(999)
bp_before = np.random.normal(loc=150, scale=15, size=20)
bp_after = bp_before - np.random.normal(loc=10, scale=5, size=20) 

# 🚨 핵심: 변화량(차이값)에 대한 정규성 검정을 수행해야 함!
diff = bp_after - bp_before

# 1. 차이값 정규성 검정
stat, p_val = stats.shapiro(diff)
print(f"[차이값 정규성 검정] p-value: {p_val:.4f}")
[차이값 정규성 검정] p-value: 0.8783
# (위배 시 비모수: stats.wilcoxon(bp_after, bp_before, alternative='less'))

# 2. 대응 표본 t-검정 (단측 검정: after가 before보다 '작은가?')
# Python scipy 1.6.0 이상부터 alternative 옵션 지원 ('two-sided', 'less', 'greater')
t_stat, t_pval = stats.ttest_rel(bp_after, bp_before, alternative='less')
print(f"[대응표본 t-검정] p-value: {t_pval:.4f}")
[대응표본 t-검정] p-value: 0.0000

📝 [결론 해석 모범 답안 작성법]

환자들의 투여 전후 혈압 차이값(투여 후 - 투여 전)에 대해 정규성 검정을 수행한 결과 정규성을 만족하였다. 약 투여 후 혈압이 더 낮아졌는지를 확인하기 위해 단측(좌측) 대응 표본 t-검정을 수행한 결과, p-value가 유의수준 0.05 미만으로 도출되어 귀무가설을 기각한다. 따라서 새로운 혈압 강하제는 환자들의 혈압을 유의미하게 하강시키는 효과가 있다고 결론 내릴 수 있다.

⛔ ADP 출제 위원회를 울리는 감점 방지 체크리스트

  1. “비모수 검정 함수 이름이 뭐였더라?”
    • 정규성을 만족하지 않을 때의 대체 함수입니다.
    • 단일/대응 표본 t-검정 -> 윌콕슨 부호순위 검정 (wilcoxon(), wilcox.test(paired=T))
    • 독립 표본 t-검정 -> 맨-휘트니 U검정 (mannwhitneyu(), wilcox.test())
  2. “등분산성 검정은 언제 건너뛰나요?”
    • 분산을 비교할 ’다른 집단’이 없는 단일 표본, 그리고 ’서로 같은 개체’인 대응 표본 t-검정에서는 절대 Levene/Bartlett 검정을 하지 않습니다. (시간 낭비 + 감점 요인)
  3. “방향 조심: Greater냐 Less냐”
    • 두 변수의 순서(A, B)를 어떻게 넣느냐에 따라 greater와 less의 의미가 완전히 반대가 됩니다. t.test(A, B) 이면 “A - B”의 차이를 의미하므로, A가 더 큰지 알고 싶다면 greater가 맞습니다. 헷갈리면 항상 양측 검정(two-sided) 도출 후, t-통계량의 부호를 보고 해석하는 것도 꿀팁입니다.
